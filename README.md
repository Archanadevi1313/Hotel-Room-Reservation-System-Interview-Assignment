# Hotel Room Reservation System

This project implements a hotel room booking system that optimally assigns rooms
based on minimum travel time (horizontal + vertical), as per the given problem statement.

## Features
- Book up to 5 rooms at a time
- Priority booking on same floor
- Optimal cross-floor booking based on travel time
- Random occupancy generation
- Reset booking
- Visual representation of hotel layout

## Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)
